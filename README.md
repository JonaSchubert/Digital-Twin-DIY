Digital Twin Workshop (Cesium)


Overview
This repository supports a hands‑on workshop that helps students build their very own digital twin in Cesium — without needing to know a single line of code.
Using prompt engineering with large language models (LLMs) and Cesium Sandcastle, students learn how to create, extend, and reason about a functioning digital twin in a modular, scalable way.
By the end of the workshop, participants will have a working digital twin that combines:

3D geospatial context
BIM / IFC models
Interactive feature‑level selection
External data (CSV from GitHub)
Real‑world sustainability metrics such as energy and carbon


Learning Goals
This workshop is designed to help students:

Understand what a digital twin actually is (beyond just a static 3D model)
Develop industry‑relevant digital twin workflows without traditional coding barriers
Build confidence working with 3D geospatial technology
Learn how BIM, GIS, and external data connect in practice
Use LLMs effectively to incrementally build software systems

No prior experience with JavaScript, Cesium, or BIM is required!
